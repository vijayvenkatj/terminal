# 🚀 Personal Portfolio

A modern, responsive, and interactive developer portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Features a **terminal-style interface**, draggable elements, and smooth animations.

## 🔥 Features
- **Terminal Interface**: Custom terminal for showcasing skills, projects, and contact details.
- **Draggable & Resizable Terminal**: Move and resize the terminal for a unique experience.
- **NeoFetch-Inspired Header**: A Linux-style system info display.
- **Dynamic Background**: Grid and particle animations for an aesthetic look.
- **Fully Responsive**: Optimized for both desktop and mobile.
- **SEO & Performance Optimized**.

## 🛠️ Technologies Used
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **UI Enhancements**: React-Draggable, React-Resizable

## 🛠️ Usage
- Drag the terminal.
- Explore skills, projects, and contact info inside the terminal.
- Enjoy a dynamic portfolio experience!
